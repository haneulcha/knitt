# Knitt Roadmap

뜨개질 패턴 DSL을 **개인 도구 → 웹 서비스 → 시뮬레이터**로 발전시키기 위한 로드맵.

---

## Phase 0: Core Engine (완료)

- [x] DSL 문법 설계 및 파서 구현 (recursive descent)
- [x] AST 기반 코 수 검증 시스템
- [x] 텍스트 렌더러 (한국어/영어)
- [x] SVG 차트 렌더러
- [x] 통계 렌더러
- [x] Result<T,E> 기반 에러 처리

---

## Phase 1: 도안 변환기

기존 코어 엔진 위에 다양한 출력 포맷을 추가하여, 하나의 패턴을 여러 스타일로 변환할 수 있게 한다.

### 렌더러 확장
- [ ] 줄글 렌더러 — 자연어 서술형 ("1단: 겉뜨기 2코, 안뜨기 2코를 5번 반복합니다")
- [ ] 일본식 도안 렌더러 — JIS 기호 기반 차트 (×, -, ○ 등 표준 기호)
- [ ] 국제 표준 기호 차트 — Craft Yarn Council 스타일 심볼

### 코어 개선
- [ ] Cast-on 구문 추가 (`Cast on 20:` 또는 `CO 20`)
- [ ] 더 많은 스티치 종류 (bind off, pick up, bobble 등)
- [ ] 에러 메시지 한국어 지원 및 위치 정보 개선
- [ ] 패턴 파일 포맷 정의 (`.knit` 확장자, 메타데이터 헤더)

### CLI
- [ ] `knitt validate <file>` — 패턴 검증
- [ ] `knitt render <file> --format chart|text|prose|jis` — 포맷 변환
- [ ] `knitt stats <file>` — 통계 출력

---

## Phase 2: 웹 도안 에디터

브라우저에서 패턴을 작성하고 실시간으로 검증/미리보기할 수 있는 플레이그라운드.

### 에디터
- [ ] Monaco Editor 기반 DSL 에디터 (구문 강조, 자동완성)
- [ ] 실시간 파싱 + 검증 피드백 (에러 인라인 표시)
- [ ] 실시간 차트 미리보기 (에디터 옆 split view)

### 변환 UI
- [ ] 출력 포맷 선택 (차트/줄글/일본식/텍스트)
- [ ] SVG/PDF 내보내기
- [ ] 공유 링크 생성 (URL에 패턴 인코딩)

### 기술 스택 (예상)
- React + Vite
- 코어 엔진을 브라우저 번들로 사용 (순수 함수이므로 그대로 동작)

---

## Phase 3: 완성작 시뮬레이터

작성한 도안을 기반으로 완성된 뜨개질 작품을 시각적으로 시뮬레이션한다.

### 2D 시뮬레이션
- [ ] 스티치 단위 렌더링 — 각 코가 실제 뜨개질처럼 엮인 모습
- [ ] 실 색상 팔레트 — 색상 지정 및 컬러워크 미리보기
- [ ] 실 질감 표현 — 울, 면, 모헤어 등 텍스처 시뮬레이션

### 3D 시뮬레이션 (장기)
- [ ] 평면 편물 → 입체 형태 변환 (원통형 뜨개질, 셰이핑 반영)
- [ ] 치수 기반 시뮬레이션 — 게이지 입력 → 실제 크기 추정

### 실 데이터
- [ ] 실 브랜드/종류 데이터베이스
- [ ] Ravelry API 연동 (실 정보, 게이지 데이터)

---

## 원칙

- 각 Phase는 독립적으로 가치가 있어야 한다 — Phase 1만으로도 유용한 CLI 도구
- 코어 엔진은 순수 함수로 유지 — 브라우저/Node.js/CLI 어디서든 동작
- Phase별로 별도 spec → plan → implementation 사이클을 거친다
